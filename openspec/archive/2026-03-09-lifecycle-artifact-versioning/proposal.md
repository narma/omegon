# Version-control lifecycle artifacts by default

## Intent

Ensure design-tree and OpenSpec artifacts used as durable project documentation are tracked in git by default, document the policy clearly, and add a repository guard that warns or fails when lifecycle artifacts are left untracked. Keep transient cleave workspace/runtime outputs out of the repo unless they are intentional project artifacts.
